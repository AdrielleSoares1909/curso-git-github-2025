git init . > cria um repositorio na pasta atual
git status > mostra os status do trabalho
git add nome_arquivo > adiciona arquivo para ser commitado
git commit -m "msg" > faz commit do que estava para ser commitado
git log > mostra todos os historicos de commits

## FLUXO DE TRABALHO

1. CRIAR O DIRETORIO DO PROJETO
2. INICIAR O GIT DENTRO DO DIRETORIO CRIADO(git init .)
3. EDITA OU CRIA ARQUIVOS
4. GIT STATUS
5. GIT ADD <NOME ARQUIVO>
6. GIT STATUS
7. GIT COMMIT -M "MENSAGEM"
8. GIT STATUS 
9. VOLTA PARA O PASSO 3.
